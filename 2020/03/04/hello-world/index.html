Hello World

所有程序员的第一篇Blog应该都是叫Hello World，2019年我辞去工作，2020年我重新出发。

我做了很多年的软件开发，也做过很多的项目，以前我特别自豪地跟别人说，我是一个全栈工程师。

在原单位，我什么都干，前台，后台，数据库，需求调研，乃至上线。

2019年年中，也就是我过生日的时候，我仔细问过自己会什么，回想了一下，我什么都不会。

好的，我决心重新出发。

学习清单

  • Linux 服务器的学习，资料是鸟哥私房菜。做饭走起。
  • JAVA 想学些别的开发语言，以前总是一知半解。
  • VIM 神器。不会用。现在也只是会一点。
  • MarkDown 个人超级喜欢，超按级好用。只会简单语法。
  • C# 老本行。不能丢。自己会的太少。
  • 思维导图。整理一些有用的东西出来。
